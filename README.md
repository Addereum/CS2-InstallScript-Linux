EN:
1. Download or clone the repository.
2. Unpack the Archive.
3. Change directory in the folder.
4. Run `chmod 777 installsrv.sh && chmod 777 updateserver.sh && chmod 777 && variables.sh`
5. Execute `./installsrv`.
